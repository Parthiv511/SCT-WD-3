# Quiz Game Application

website link: https://sct-wd-3-tjff.onrender.com/
## 📌 Project Overview

A modern and responsive Quiz Game Application built using HTML, CSS, and JavaScript.

The application allows users to:

- Answer multiple-choice questions
- Track quiz progress
- View countdown timer
- Get instant answer feedback
- Calculate final score
- Restart the quiz

The project features a premium glassmorphism UI with smooth animations and responsive design.
